# Electron React Webpack

Clone to try a simple Electron app using webpack and React. 

## Develop and deploy

### Install

```bash
$ cd path/to/repo
$ npm install
$ npm start
```

### Develop

1. Start Webpack-dev-server

```bash
$ npm run dev-file
```

2. Start Electron

```bash
$ npm run dev-app
```

### Deploy

1. Deploy js html and css files

```bash
$ npm run deploy-file
```

2. Start Electron

```bash
$ npm run deploy-app
```
